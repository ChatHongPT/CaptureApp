# CaptureApp

- 원하는 포즈의 사진 추가
- 사진 포즈 분석
- 분석된 포즈 화면에 표시

## 커밋 컨벤션

- 형식: <타입> 설명

(예시 1) feat: 사진 업로드 기능 추가
(예시 2) fix: 사진 검색 기능 버그 수정

### 타입(Type)
- `feat`: 새로운 기능 추가
- `fix`: 버그 수정
- `docs`: 문서 변경
- `style`: 코드 스타일 변경(공백, 포맷팅 등)
- `refactor`: 코드 리팩토링
- `test`: 테스트 코드 추가/수정
- `chore`: 빌드 프로세스 수정 등의 보편적인 작업

### 설명(Description)
변경 사항에 대한 설명
